# Predefined functions

## Utility

```nbt
fn unit_of<T>(x: T) -> T
fn value_of<T>(x: T) -> Scalar
```

## Math

### Basics

```nbt
fn abs<T>(x: T) -> T
fn round<T>(x: T) -> T
fn floor<T>(x: T) -> T
fn ceil<T>(x: T) -> T
fn mod<T>(a: T, b: T) -> T
fn sqrt<D>(x: D^2) -> D
fn sqr<D>(x: D) -> D^2
```
